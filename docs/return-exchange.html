RETURNS & EXCHANGES

7-DAY RETURN POLICY

We accept returns within 7 days of delivery. Items must be unworn, unwashed, and with original tags attached.

  • Initiate return via our online form or email.
  • Use original packaging to ship the item back.
  • Refund (excluding shipping) will be processed within 5 business days after inspection.

HOW TO EXCHANGE

  • Select "Exchange" in the return form and specify desired size/color.
  • If the requested item is available, we'll ship it immediately after receiving the original.
  • Price differences will be adjusted; you'll only pay if the new item costs more.

NON‑RETURNABLE ITEMS

  • Gift cards
  • Underwear & lingerie
  • Face masks (hygiene)
  • Final sale / clearance
  • Customized items
  • Swimwear without liner

If you receive a defective or wrong item, please contact us immediately — we'll make it right.
